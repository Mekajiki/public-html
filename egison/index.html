What is Egison?

Egison is a functional programming language.
A feature of Egison is the strong pattern match facility.
With Egison, you can represent pattern matching for non-inductive data intuitively,
such as set, multiset, or algebraic structure such as group, ring, field, and so on.

Egison's other features are

  • Lazy evaluation

How to install?

Egison is implemented in Haskell.
Egison is distributed as a hackage program.

To install Egison, you need to install GHC and Cabal-install.
If you install Haskell Platform, both of GHC and Cabal-install will be installed.

After you installed Hasekll Platform, perform following commands.

% cabal update
% cabal install egison
If successfully installed, there will be an output where the binary file is generated.
Please add new path to $PATH variable of your shell, to make your shell to find egison command.
(If your OS is Windows, you don't need to set the path of the shell.)

After that, you can use follwing command.

% egison

How to update?

You can update Egison with follwing commands.

% cabal update
% cabal install egison
