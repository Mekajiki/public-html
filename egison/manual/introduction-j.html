イントロダクション

とりあえず，Egisonを動かしてみましょう．
この章では，ポーカーの役判定プログラムを書いて動かしてみます．
トランプのカードのコレクションを多重集合としてパターンマッチするプログラムです．
コレクションデータのパターンマッチの表現は，Egisonの得意とするものです．
これを動かしてみて，Egisonがどういうものなのか感じてみてください．

インストールと実行

EgisonはHaskellを使って実装されています．
EgisonはHackageのパッケージとして配布されています．

Egisonをインストールするには，GHCと Cabal-installがインストールされている必要があります．
Haskell Platformをインストールすれば， その両方が自動でインストールされます．

Hasekll Platformをインストールしたら，コマンドラインで以下のコマンドを実行してみてください． 

    
  % cabal update
  % cabal install egison
もし，インストールが成功したら，実行ファイルが生成されたパスが出力されるはずです．
シェルが実行ファイルを見つけれるように，シェルの$PATH変数にそのパスを追加してください．
(Windowsの場合はパスの設定が必要ないようです)

そうすると，以下のコマンドが実行できるはずです． 

    
  % egison

アップデートの方法

以下のコマンドで最新版にアップデートできます． 

    
  % cabal update
  % cabal install egison

インタプリタを起動しよう

インストールに成功すると，コマンドラインでegisonと打つとEgisonのインタプリタが起動します．
Control-Dを打つとインタプリタを終了できます．(ControlキーとDキーを同時押し) 

    
  % egison
  Egison, version X.X.X : http://hagi.is.s.u-tokyo.ac.jp/~egi/egison/
  Welcome to Egison Interpreter!
  > ^D
  Leaving Egison.
  Byebye. See you again! (^^)/

いくつか，Egisonに入力を与えてみましょう． 

    
  > (define $x 2)
  x
  > (test (+ x 3))
  5
  > (define $f (lambda [$x $y] [(+ x y) (* x y)]))
  f
  > (test (f x 4))
  [6 8]
この例では，2つのトップ式(top expression)がでてきました．
define式とtest式です．
define式は1番目の引数の変数の2番目の引数の式の値への束縛をトップレベル環境に加えます．
test式は1番目の引数の式を評価しその値を表示します．
